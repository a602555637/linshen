# Markdown 基本语法
## 标题
  在想要设置为标题的文字前面加#来表示
  >+ #一级标题
  >+ ##二级标题
  >+ ######六级标题
## 字体样式
  + **加粗**
  + *斜体*
  + ***斜体加粗***
  + ~~删除线~~
## 引用
  > 参考标题处内容
  >> 仅供参考
## 分割线
  ***
  *** 或者 *****
  *****
## 图片
  ![avatar](/img01.jpg "way back!")
## 超链接
  * [Github](http://www.github.com/a602555637 "github")
  * <a href="http://www.github.com/a602555637" target="_blank">简书</a>

## 列表
  - 无序列表-
     + 无序列表+，嵌套需要敲三个空格
  * 无序列表*
   1. 有序列表1
      1. 有序列表1，嵌套需要敲三个空格
   2. 有序列表2
## 表格
  ```
  表头|表头|表头
  ---|:--:|---:
  内容|内容|内容
  内容|内容|内容

  第二行分割表头和内容。
  - 有一个就行，为了对齐，多加了几个
  文字默认居左
  -两边加：表示文字居中
  -右边加：表示文字居右
  注：原生的语法两边都要用 | 包起来。此处省略
  ```

  姓名|技能|排行
  --|:--:|--:
  刘备|哭|大哥
  关羽|打|二哥
  张飞|骂|三弟
