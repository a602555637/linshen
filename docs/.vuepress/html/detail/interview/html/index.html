# html

# <!DOCTYPE>什么意思

告知 浏览器文档使用哪种 HTML 或 XHTML 规范，告诉浏览器按照何种规范解析页面

# HTML 语义化

当页面样式加载失败的时候能够让页面呈现出清晰的结构 有利于 seo 优化，利于被搜索引擎收录（更便于搜索引擎的爬虫程序来识别） 便于项目的开发及维护，使 html 代码更具有可读性，便于其他设备解析
